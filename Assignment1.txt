---Review Question---
Q1  : How would you define Machine Learning?
Ans : Machine Learning adalah proses pembelajaran suatu mesin untuk mendapatkan output yang diinginkan tanpa penggunaan instruksi spesifik.

Q2  : Would you frame the problem of spam detection as a supervised learning or an unsupervised learning problem?
Ans : Menurut saya spam detection adalah masalah yang diselesaikan dengan menggunakan supervised learning.

Q3  : What are the two most common supervised tasks?
Ans : Object Recognition dan speech recognition.

Q4  : Can you name four common unsupervised tasks?
Ans : Clustering, anomaly detection, dimensionality reduction, dan finding association rules.
